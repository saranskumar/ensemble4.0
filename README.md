This project is a web-based registration system created for **IEEE SCT SB** to manage participant registrations for the event **Ensemble 4.0**.

It allows users to register online, receive a unique event token, and join the official event group after successful registration.

---

## Event Details

**Organizer:** IEEE SCT SB  
**Event Name:** Ensemble 4.0  


---

## What This Project Does

- Collects participant details through an online form
- Confirms registration instantly on the same page
- Generates a unique event token for each participant
- Displays the token as a digital ticket
- Instructs users to take a screenshot of the ticket
- Provides a WhatsApp group link after registration
- Stores all registration data safely in Google Sheets

---

## Registration Confirmation

After successful registration, the user sees:

- **Registration Confirmed** message  
- A unique event token (example: `ES4091`)  
- Message confirming email notification  
- Instruction to screenshot the ticket  
- Button to join the official WhatsApp group
